# 🛡️ 2026年度~ コンプライアンス・リスクマネジメント教育計画案

## 1. プログラム概要

- **目的:** 「知らなかった」による不祥事を防ぎ、社員と会社を守る文化を作る。
- **重点項目:** 情報漏洩対策、SNS適正利用、新入社員の早期意識付け。
- **実施形式:**
    - **毎月:** 全社会/会議での5分間スピーチ。
    - **動画:** スピーチ内容を凝縮した2〜3分の解説動画（アーカイブ化）。
    - **テスト:** AI（ChatGPT等）を活用した理解度チェック。

# 4. 新入社員向け：コンプライアンス・クイックスタートガイド

*入社初日に必ずチェックさせる「最優先」コンテンツです。*

1. **SNSの鉄則:** 「会社の制服・バッジ・オフィス・PC」を写した投稿は、内容に関わらず原則禁止。
2. **相談の鉄則:** 「あれ、これマズいかも？」と思ったら、自分で解決しようとせず、10分以内に上司へ一報。
3. **パスワードの鉄則:** 付箋でPCに貼らない。他人に教えない。
4. **持ち出しの鉄則:** 許可なく社外に情報を持ち出さない（個人メールへの転送含む）。

## 5. 【プロのアドバイス】自社制作で成功させるコツ

1. **「専門用語」を捨てる**
    - 「善良なる管理者の注意義務（善管注意義務）」と言うより、「借りている道具を、持ち主以上に大切に扱う責任」と言い換えた方が、動画やスピーチでは伝わります。
2. **「恐怖」ではなく「連帯感」**
    - 「やったらクビだぞ」と脅すのではなく、「一人のミスで、全員のボーナスや会社の信頼がなくなる。だから助け合おう」という伝え方をしてください。
3. **AIテストの運用**
    - Notionの「データベース」機能で、テストの回答結果を集計できるようにしておくと、どの部署の理解度が低いか可視化でき、次月のスピーチの参考になります。

コンプライアンス教育を内製化する際、ゼロから資料を作るのは「プロの仕事」としては非効率です。「公的機関や信頼できる団体が公開している『お墨付き』の素材を、自社風にアレンジする」のが、最も低コストでクオリティを担保できる勝ちパターンです。

リサーチの結果、Notionに埋め込んだり、5分スピーチのネタにしたりするのに最適な「神教材」を厳選して報告します。

## 1. 情報漏洩・セキュリティ対策の鉄板（IPA）

情報処理推進機構（IPA）の資料は、図解が豊富でそのまま動画の絵コンテに使えます。

| **サイト名** | **特徴・活用方法** |
| --- | --- |
| [**IPA 映像で知る情報セキュリティ**](https://www.ipa.go.jp/security/keihatsu/videos/index.html) | 1〜3分の短編動画が豊富。朝礼で流すのに最適。 |
| [**中小企業の情報セキュリティ対策ガイドライン**](https://www.ipa.go.jp/security/guide/sme/about.html) | 「自己診断チェックリスト」が付属しており、AIテストの設問作りに直結します。 |
| [**5分でできる！情報セキュリティ自社診断**](https://www.ipa.go.jp/security/guide/sme/check.html) | 毎月のスピーチの後に、その場でスマホで回答してもらう運用が可能です。 |

## 2. ハラスメント対策の宝庫（厚生労働省）

セクハラ・パワハラに関しては、厚労省の特設サイトが最も網羅的です。

- [**あかるい職場応援団（厚労省）**](https://www.no-harassment.mhlw.go.jp/)
    - **活用法:** 「動画で学ぶハラスメント」コーナーに、ドラマ形式の教材があります。
    - **ここがプロの視点:** 「裁判例」のデータベースが非常に優秀です。5分スピーチで「実際にこれだけの損害賠償が出ました」と事実を伝えるだけで、説得力が段違いに上がります。

    - マンガ形式の解説があり、新入社員でも飽きずに読めます。Notionのページに画像を貼り付けて、「ここがNGポイントです」と解説を入れるのがおすすめです。

## 5. 【実務テクニック】Notion × AIテストへの繋げ方

これらの教材を見つけたら、以下の手順で「自走」させます。

1. **PDFをダウンロード:** 上記サイトのガイドラインや事例集（PDF）を入手。
2. **AIに読み込ませる:** ChatGPTなどのAIにそのPDFをアップロード。
3. **プロンプトを入力:**
    
    > 「このPDFの内容に基づいて、新入社員が絶対に間違えてはいけないポイントを3問、3択テスト形式で作成して。解説は、当社の行動指針である『誠実さ』を強調した文章にして。」
    > 
4. **Notionへ:** 生成されたテストをNotionのインラインデータベースやGoogleフォームに貼り付け。

### プロのアドバイス

ネット上の教材は「一般的すぎる」のが弱点です。

「IPAの動画ではこう言っていますが、**わが社のオフィス環境だと、あの共有棚が一番のリスクですよね**」といった具合に、**「一般論（ネット教材）＋自社固有の具体例」**をセットで話すことで、社員の「自分事化」が劇的に進みます。
